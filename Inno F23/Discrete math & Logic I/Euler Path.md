## Definitions
- A [[Walk|path]] is called Euler, if it contains every edge and only once.
- A [[Graph]] having an Euler path is called *semi-Eulerian*
- A closed Euler path is an [[Euler Cycle]]
## Remark
- An Euler path = An Euler trail
