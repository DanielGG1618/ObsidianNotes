## Definition
- A [[Sequence]] $a_0, a_1, ...$ is called a linear recurrence (recursive) sequence (of order $k$), if $$\displaystyle a_{n+k} = t_1a_{n+k-1}+...+t_ka_n=\sum^k_{i=1}t_ia_{n+k-i}$$
where all $t_i$ are fixed coefficients
## Example
$$a_{n+2} = a_{n+1}+a_n$$
