## Definitions
- A [[Walk|cycle]] is called Hamilton, if it contains every vertex and only once
- A [[Graph]] having an Hamilton cycle is called an *Hamiltonian*
## Remark
- A Hamilton cycle = A Hamilton circuit = A Hamilton tour
